# 分拣标签（Triage Labels）

技能以五种规范分拣角色表述。本文件将这些角色映射到本仓库问题跟踪里的 **实际标签字符串**。

| mattpocock/skills 中的角色 | 本跟踪系统中的标签 | 含义 |
| -------------------------- | ------------------ | ---- |
| `needs-triage`             | `needs-triage`       | 维护者需评估此工单 |
| `needs-info`               | `needs-info`         | 等待报告者补充信息 |
| `ready-for-agent`          | `ready-for-agent`    | 规格齐备，可由 AFK 智能体认领 |
| `ready-for-human`          | `ready-for-human`    | 需人类实现 |
| `wontfix`                  | `wontfix`            | 不予处理 |

当技能提到某角色（例如「应用 AFK 就绪分拣标签」）时，使用本表右侧对应字符串。
