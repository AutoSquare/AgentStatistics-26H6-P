# AgentStatistics 开发说明

项目概述见 [README.md](../README.md)。

持久化路线：**memory**（`memory` = 记忆快照；`archive` = ZIP 工程包）

## 目录职责

| 路径 | 职责 |
|------|------|
| `Themes/` | DefaultTheme 色板与全局样式 |
| `Resources/` | 图标/背景/字体（ASCII 路径） |
| `Properties/` | Settings.settings / Resource.resx |
| `ViewModel/`、`Services/`、`Model/` | MVVM + Python 桥 |
| `ASPy/`、`ASEnv/` | Python 脚本与运行时 |
| `WebClient/` | Vue + Vite + ECharts 仪表盘源码；构建产物复制到 WPF 输出目录 |

## Codex 统计链路

- Codex 会话定位、解析、缓存、聚合、额度、费用与 CSV 字段语义以 `../CodexScope/generate_codex_data.go` 和 `../CodexScope/app.ts` 为行为真相源。
- 第一阶段不再使用 Go；`ASPy/codex_usage_stats.py` 迁移 CodexScope 核心逻辑并输出前端 JSON。
- 默认会话目录为 `%USERPROFILE%/.codex/sessions`，用户可在界面修改，保存键为 `codexSessionsPath`。
- 增量缓存写入 `%AppData%/AgentStatistics/codex_usage_cache.json`，不写入仓库。
- WPF 主窗口通过 WebView2 加载 `WebClient/dist/index.html`，并通过 WebView message 发送设置、刷新状态与 Codex 统计结果。
- 自动刷新由 WPF 监听 sessions 目录下 `*.jsonl` 文件变化触发；手动刷新按钮保留。
- 隐私边界：只处理 Token、模型、时间戳、会话 id、cwd basename、rate limit、耗时、失败状态等元数据；不读取或导出提示词、助手正文、工具输出或文件内容。

## UI/UX 约束

- 任何涉及 WPF 或前端界面结构、组件、图表、导航、空状态、视觉设计或交互改动的任务，先调用 `ui-ux-pro-max`。
- 当前设计系统基线：Data-Dense Dashboard，浅色专业仪表盘，主色 `#1E40AF`，强调色 `#D97706`，图表使用 ECharts。
- 交付前至少检查：文本对比度、焦点态、图表空状态、加载/失败状态、按钮禁用态、375/768/1024/1440 宽度无重叠。

## memory 模式

- `%AppData%/AgentStatistics/user_settings.json` — 窗口布局（WindowPlacement：宽高、位置、最大化状态）
- `%AppData%/AgentStatistics/session_snapshot.json` — 上次会话业务表 + `uiState.statusText`（状态栏文案）
- 无 `{Ext}` 工程文件菜单
- 契约与接线：Skill 内 `references/ui-state-contract.md`、`references/persistence-wiring.md`

### uiState 扩展键（业务登记）

| 键 | 类型 | 说明 |
|----|------|------|
| `statusText` | string | 模板必选；由 MainWindowViewModel 写入 |

## archive 模式

- 工程包扩展名：`.proj`
- `Assets/Baseline/` — 新建工程种子表
- Export → Driver → Merge → Save/Open（**Ctrl+S**、文件菜单）
- 窗口布局仍写入 `user_settings.json`
- **文件关联**：启动时 `FileAssociationUtilities.TryRegisterOnStartup()` 写入 `HKCU\Software\Classes\.proj`，open 命令 `"exe" "%1"`
- **双击打开**：`App.xaml` 使用 `Startup="Application_Startup"`（**非** `StartupUri`）；`e.Args` 中 `{Ext}` 路径优先于「最近工程」
- Debug/Release 或输出目录变更后须 **再 F5 一次** 刷新关联
- 图标：应用图标使用 `Resources/icons/logo.ico`；前端品牌标识使用 `WebClient/public/logo.png`，来源为 `Resources/icons/logo.png`。

## 构建

1. `setup_python_env.py` 创建 `ASEnv/`
2. `cd WebClient && npm install && npm run build`
3. `dotnet build`
4. `py -3 ASPy/test_codex_usage_stats.py`
5. `smoke_bridge.py` 烟测（需要 Python 环境时）

## 切换持久化路线

```powershell
py -3 scripts/migrate_persistence_mode.py --project-dir . --abbr AS --from memory --to archive --import-last-snapshot --apply
```
